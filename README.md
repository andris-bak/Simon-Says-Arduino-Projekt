# Simon-Says-Arduino-Projekt
Egy Arduino projekt mely a közismert Simon Says játékot hivatott imitálni.
A játék legelején felvillanak a LED-ek és miután egy gombot lenyomtunk máris játszhatunk. Cél hogy minél többször tudjuk a megadott szekvenciát visszaadni a játéknak.
Ha vesztettünk, akkor véget ér a játékunk a "High_Score-t" elmenti. Ezt, ha megdöntjük átíródik.
